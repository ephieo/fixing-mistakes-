DO KYUNGSOO

I struggled a lot with figuring out how to implement keyboard navigation and how to make the carousel pause. While reading up more on event listeners and DOM manipulation I managed to achieve it by creating an if statement while using the events keycode, mouseleave and mouseenter. I found the book “JavaScript and JQuery: Interactive Front-End Web Development” by Jon Duckett to be quite helpful, just my luck that it arrived the day after the application submissions. Although there are some limitations to the book seeing as it was published the year that ES6 was released so some things are either outdated or not mentioned. Nonetheless I found it quite helpful for understanding the basics of nodelists, the DOM tree etc..

I also struggled with which method to use to create the carousel. I watched many videos about how people attempted creating a slideshow but many didn’t use vanilla javascript or only achieved parts of the carousel requisites. My aim was to find a method that I could replicate and understand. I initially approached the task by ordering my slides into an array and iterating through the array by moving a distance of slide width while using click events. However with the array approach I couldn’t figure out how to loop the carousel so I used a clone approach. Using the cloneNode method I created a first and last clone place them at the beginning and end of my slides list and looped through the slides by using setInterval and the transitionend event. Once the transition ended I could then return it to the first or last clone.
